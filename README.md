# SAFARNAMA

Run this app on your machine in minutes.

## Requirements
- Node.js installed
- Expo Go app on your phone (Android/iOS)

## Setup
```bash
git clone https://github.com/zenik-kun/safarnama.git
cd safarnama
npm install
npx expo start -c
```

## Use
- Make sure your phone and computer are on the same Wi‑Fi.
- Open Expo Go and scan the QR code from the terminal/devtools.
- The app will load and hot‑reload on save.
